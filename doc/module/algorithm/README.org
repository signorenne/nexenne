#+TITLE: nexenne::algorithm
#+OPTIONS: toc:nil num:nil ^:nil
#+STARTUP: showall

Header-only C++23 algorithms that are not in the standard library but are
broadly useful for embedded and systems work, built on
[[file:../container/README.org][nexenne::container]]. Grouped by category: binary search variants, integer
sorts, string matching, numerical routines, hashing, checksums, byte
encodings, and graph algorithms.

Every category has a themed article below and a runnable example under
=examples/algorithm/=. Each routine is exhaustive-tested (known-answer vectors
and differential cross-checks) and reports recoverable failures through
=std::expected=, never exceptions.

* Categories

- [[file:binary-search.org][binary search]] : =find_sorted=, =exponential_search=,
  =interpolation_search= (index-returning variants beyond =std::ranges=).
- [[file:sort.org][sort]] : =counting_sort=, =radix_sort= (linear-time, stable integer sorts).
- [[file:string-matching.org][string matching]] : =kmp=, =z_function=, =boyer_moore=, =aho_corasick=,
  =levenshtein=, =suffix_array=.
- [[file:numerical.org][numerical]] : compensated sums and online stats, root finding, integration,
  interpolation, ODE integrators, FFT.
- [[file:hashing.org][hashing]] : =fnv1a=, =murmur3=, =xxhash= (non-cryptographic; hash tables,
  fingerprints, content addressing).
- [[file:checksum.org][checksum]] : =adler32=, =crc= (integrity checks; generic Rocksoft-model
  CRC engine with named presets).
- [[file:encoding.org][encoding]] : =hex=, =base64=, =base32=, =url=, =cobs= (byte-to-text and
  byte-stuffing codecs; heap-free and heap-allocating overloads).
- [[file:graph.org][graph]] : traversal (=bfs=, =dfs=), shortest paths (=dijkstra=, =a_star=,
  =bellman_ford=, =floyd_warshall=), structure (=topological_sort=,
  =connected_components=, =tarjan_scc=, =kruskal_mst=, =lca=); these reuse
  =container::graph= and friends.
