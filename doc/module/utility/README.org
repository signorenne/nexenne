#+TITLE: nexenne::utility
#+OPTIONS: toc:nil num:nil ^:nil
#+STARTUP: showall

The foundation module: small, focused, header-only C++23 utilities with no
dependencies, the layer everything else in nexenne is built on. Each fills a gap
the standard library leaves open, and each is independent, so take only the ones
you need. They are grouped below into eight themes; every entry links to its guide
(purpose, the idea behind it, idiomatic usage) and most to a runnable example under
[[file:../../../examples/utility/][examples/utility/]] that you can build
(=NEXENNE_BUILD_EXAMPLES=ON=) and run to see it work.

Include a single header (=nexenne/utility/<name>.hpp=) or the umbrella
=nexenne/utility/utility.hpp=. Everything lives in =nexenne::utility=.

* Callables

- [[file:overloaded.org][overloaded]] : build a visitor from a set of lambdas (for =std::visit=).
- [[file:function_ref.org][function_ref]] : non-owning, type-erased callable view (a parameter type).
- [[file:in_place_function.org][in_place_function]] : owning, heap-free, fixed-capacity callable.

* Ranges and values

- [[file:for_each_non_null.org][for_each_non_null]] : apply a callable to each non-null element of a range.
- [[file:discard.org][discard]] : evaluate and discard a value (replaces void casts).

* Scope and resource management

- [[file:defer.org][defer]] : run an action at scope exit.
- [[file:scope_guard.org][scope_guard]] : like =defer=, but dismissable (commit/rollback).
- [[file:unique_resource.org][unique_resource]] : generic RAII for any handle and its releaser.
- [[file:lazy.org][lazy]] : thread-safe compute-once-on-first-access value.

* Numeric, bit, and hashing

- [[file:narrow_cast.org][narrow_cast]] : checked narrowing conversion (asserts in debug).
- [[file:bit.org][bit]] : bit manipulation beyond =<bit>= (reverse, masks, fields).
- [[file:align.org][align]] : power-of-two alignment for sizes and addresses.
- [[file:hash.org][hash]] : Boost-style =hash_combine= primitives.

* Bytes and buffers

- [[file:endian.org][endian]] : read and write integers in big- or little-endian order.
- [[file:buffer_cursor.org][buffer_cursor]] : bounds-tracked read/write position in a byte span.

* Distinct types

- [[file:strong_typedef.org][strong_typedef]] : opt-in strong type wrapping a value.
- [[file:non_null.org][non_null]] : pointer wrapper that enforces a non-null contract.
- [[file:flags.org][flags]] : type-safe bitmask over a scoped enum.
- [[file:static_string.org][static_string]] : compile-time string usable as a template parameter.

* Compile-time metaprogramming

- [[file:type_list.org][type_list]] : a list of types with the usual metafunctions.
- [[file:meta.org][meta]] : =type_name=, =function_traits=, =always_false_v=.

* Reflection, errors, diagnostics

- [[file:enum_to_string.org][enum_to_string]] : compile-time enum reflection.
- [[file:expected_utils.org][expected_utils]] : ergonomic helpers for =std::expected=.
- [[file:source_location_format.org][source_location_format]] : format a source location into a buffer.
