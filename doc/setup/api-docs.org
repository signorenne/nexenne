#+TITLE: API documentation
#+OPTIONS: toc:nil num:nil ^:nil
#+STARTUP: showall

Public API docs are Doxygen, generated from the headers.
[[file:../../cmake/nexenne_doxygen.cmake][nexenne_doxygen.cmake]] runs
=find_package(Doxygen)=, and when Doxygen is missing and
=NEXENNE_FETCH_DOXYGEN_IF_MISSING= is ON (its default), fetches and builds the
tag in =NEXENNE_DOXYGEN_GIT_TAG= (default =Release_1_12_0=) from source so a
fresh clone is self-contained. Opt out of the source build with
=-DNEXENNE_FETCH_DOXYGEN_IF_MISSING=OFF=; if Doxygen is then unavailable the doc
targets are skipped with a warning rather than created.

* Build

Docs are off by default; configure with =NEXENNE_BUILD_DOCS=ON= to define the
targets.

#+BEGIN_SRC sh
cmake -B build/docs -DNEXENNE_BUILD_DOCS=ON
cmake --build build/docs --target nexenne_docs            # every module
cmake --build build/docs --target nexenne_<name>_docs     # one module
#+END_SRC

=nexenne_docs= is the aggregate target; each module that has a =doc/=
subdirectory contributes a =nexenne_<name>_docs= target via
=nexenne_add_module_docs(<name>)= (called from
=modules/<name>/doc/CMakeLists.txt=). HTML output lands under
=build/docs/modules/<name>/doc/html/index.html=.

* Doxyfile

Each module's Doxyfile is generated by =configure_file= from a template: the
module's own =doc/Doxyfile.in= if present, otherwise the shared
[[file:../../template/module/doc/Doxyfile.in][template/module/doc/Doxyfile.in]].
The template is Javadoc-flavored (=JAVADOC_AUTOBRIEF=, first sentence is the
brief), takes the module's =include/= as input, documents only public items
(=EXTRACT_ALL = NO=, =EXTRACT_PRIVATE = NO=), and strips C++23 attributes such
as =[[nodiscard]]= from signatures via =PREDEFINED=.

* What the docs build enforces

The template sets =WARN_AS_ERROR = FAIL_ON_WARNINGS=, so the build fails on
Doxygen documentation /errors/: a =@param= that does not match the signature, a
=@return= on a =void= function, a broken reference. =WARN_IF_UNDOCUMENTED= and
=WARN_NO_PARAMDOC= are on. The [[file:ci.org][CI]] =docs= job runs this gate.

It does /not/ guarantee completeness: with =EXTRACT_ALL = NO=, Doxygen simply
omits an undocumented symbol rather than failing. "Every public symbol has a
Doxygen block" is therefore a review-checklist item, not an automated gate.
